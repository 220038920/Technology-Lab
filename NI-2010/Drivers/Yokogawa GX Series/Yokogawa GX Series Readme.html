Yokogawa GX/GP/GM Series Instrument Driver Readme

To rate this driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Yokogawa Electric Corporation
Supported Language(s): LabVIEW
Supported Model(s): Yokogawa GX10, GP10, GX20, GP20, GM10
Model(s) Tested: Yokogawa GM10, GX20
Interface(s): LAN, RS-232C, RS-422A, USB, Bluetooth
Firmware Revision Tested: R4.02.01

Certified: No
NI Supported: No
Source Code Available: Yes

Driver Revision: 4.2.1
Original Release Date: 06/2013
Current Revision Date: 06/2018

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 5.0 or later

For more details, refer to the LabVIEW Help under Help Å‚ VI, Function, & How-To Help.

3. Known Issues


Configure Calib Factor (IO) and Configure Calib Linear (IO) VIs connector pane patterns were changed to 5x3x3x5 from previous 4x2x2x4.
